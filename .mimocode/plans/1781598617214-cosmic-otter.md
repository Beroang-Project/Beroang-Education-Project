# Tambahkan Stat Cards dengan Judul ke Dashboard

## Perubahan
Ganti mini stats inline (text kecil di Welcome section) dengan **4 stat cards** yang lebih jelas menggunakan `Card` component.

## Layout
```
┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│ icon     │ │ icon     │ │ icon     │ │ icon     │
│ 30       │ │ 4        │ │ 3        │ │ 2        │
│ Total    │ │ Total    │ │ Tugas    │ │ Roadmap  │
│ Siswa    │ │ Mapel    │ │ Aktif    │ │ AI       │
└──────────┘ └──────────┘ └──────────┘ └──────────┘
```

- Grid `grid-cols-4 gap-4` di bawah Welcome section
- Setiap card: icon (40×40 rounded bg) + angka besar + label
- Angka tetap ditampilkan meskipun 0
- Menggunakan `Card` + `CardContent` yang sudah ada
- GSAP `.stat-card` stagger animation

## File: `app/(dashboard)/dashboard/page.tsx`
1. Hapus mini stats inline dari Welcome section
2. Tambahkan grid stat cards setelah Welcome
3. Import `TrendingUp` untuk icon Rata-rata Nilai (hitung dari assessmentStore)
4. Tambahkan `calculateFinalGrade` untuk menghitung avg grade
5. Tambahkan stat: Total Siswa, Total Mapel, Tugas Aktif, Roadmap

## Verifikasi
- `npm run build` tanpa error
- 4 stat cards terlihat dengan angka yang benar
- Angka 0 tetap ditampilkan
